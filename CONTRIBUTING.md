# Contributing

Thanks for considering a contribution.

## Good first issues

- Add Firefox support
- Add a settings page for screen color
- Improve accessibility labels
- Add automated browser tests

## Local development

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.
4. After editing files, click the extension reload button in `chrome://extensions`.

## Pull requests

Please keep pull requests focused and include:

- What changed
- Why it changed
- How you tested it

This project avoids telemetry, external services, and broad host permissions.
