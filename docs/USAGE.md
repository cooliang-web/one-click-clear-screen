# Usage

## Install from source

1. Download this repository.
2. Open `chrome://extensions` in Chrome or Edge.
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select the repository folder.

## Clear the current tab

Click the extension button in the browser toolbar. The active page is covered by a blank screen.

## Restore the page

Use any of these actions:

- Click the extension button again.
- Click the blank screen.
- Press `Esc`.

## Keyboard shortcut

The default shortcut is:

- Windows/Linux: `Ctrl+Shift+K`
- macOS: `Command+Shift+K`

You can change shortcuts from `chrome://extensions/shortcuts`.

## Expected limitations

One Click Clear Screen covers the current browser tab only. It does not hide:

- Other browser windows
- Desktop notifications
- System overlays
- Browser UI such as the address bar or toolbar
- Content in other applications

Use operating-system screen locking when you need full device privacy.
