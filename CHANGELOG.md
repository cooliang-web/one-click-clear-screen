# Changelog

## 0.3.0

- Changed the Windows executable from a full-screen overlay to a compact interface.
- The compact window is about 30% of the main screen area.
- Added the visible title `一键清屏` at the top of the window.

## 0.2.0

- Changed the Windows executable from a black cover to a clean desktop-background overlay.
- Desktop icons and windows are hidden behind the overlay while the wallpaper-style background remains visible.
- Kept `Esc` and click-to-restore behavior.

## 0.1.0

- Initial Manifest V3 browser extension
- Added one-click clear screen overlay
- Added toolbar button and keyboard shortcut support
- Added local installation documentation
- Added privacy model, maintenance notes, and contribution templates
- Added manual browser test matrix
- Added Windows executable source and build script
