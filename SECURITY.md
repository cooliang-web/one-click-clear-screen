# Security Policy

## Supported versions

Security fixes are accepted for the latest version in the `main` branch.

## Reporting a vulnerability

Please open a private security advisory on GitHub if available. If private advisories are not enabled, open an issue with a short summary and avoid posting sensitive exploit details.

## Security model

One Click Clear Screen is designed to avoid collecting data.

- No analytics
- No remote API calls
- No background page content scanning
- No broad host permissions
- Only the active tab is touched after a user action

The extension overlays a blank screen on the active page. It does not claim to lock the computer, protect operating-system notifications, or hide content outside the current browser tab.
